# Prime Number Checker

## Project Overview

This beginner-friendly Python project checks whether a number entered by
the user is Prime or Not Prime using loops and conditional statements.

## Objective

Practice: - Variables - User Input - Type Conversion - Arithmetic
Operators - Modulo Operator (`%`) - `for` Loop - `if` Statement

## Technologies

-   Python 3
-   Jupyter Notebook / Google Colab

### Libraries Used

-   No external libraries are required.

## Features

-   Accept a number from the user.
-   Check whether it is Prime or Not Prime.
-   Display the result.

## How to Run

1.  Open the notebook.
2.  Run all cells.
3.  Enter a number.

## Concepts Covered

-   Variables
-   User Input
-   Type Conversion
-   Arithmetic Operators
-   Modulo Operator (`%`)
-   `for` Loop
-   `if` Statement

## Learning Outcomes

-   Understand prime numbers.
-   Apply loops and conditions.
-   Improve problem-solving skills.
